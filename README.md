# SafeMaps – Women Safety Mapping App 🗺️🚨

> *"Safety begins with awareness and action."*

SafeMaps is a full-stack web application that allows users to report danger incidents, analyze route safety, and track travel & danger history using real-time interactive maps.

---

## ✨ Features

🚨 **Report Incidents**

* Users can report safety incidents (stalking, harassment, etc.)
* Location-based reporting with categories

🗺️ **Live Map Visualization**

* View reported incidents on interactive maps
* Real-time location plotting using Leaflet

🛣️ **Route Safety Analysis**

* Plan routes between locations
* Get safety score & nearby incident insights

📊 **Travel & Danger History**

* Track previously reported incidents
* View detailed logs with date, type, and description

---

## 🛠️ Tech Stack

**Frontend**

* React.js
* CSS (Glassmorphism UI)

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Maps**

* Leaflet.js
* OpenStreetMap

## ⚙️ Installation

```bash
git clone https://github.com/Tanishka-Darshankar/SafeMaps.git
cd SafeMaps
```

### ▶️ Run Frontend

```bash
cd client
npm install
npm start
```

### ▶️ Run Backend

```bash
cd server
npm install
node index.js
```
